# 프로젝트 설명
Javascript Es6 문법 및 알고리즘 공부 프로젝트 입니다.
<br /><br />

## JavaScript ES6 ~ ES9
- let, const
- 화살표 함수 (Arrow Function)
- 템플릿 리터럴 (Template literals)
- 클래스 (Classes)
- 스프레드 연산자(Spread operator)
- 객체 리터럴 (Object Literal)
- 프로미스 (Promises)
- 구조 분해 할당 (Destructuring assignment)
- 기본 매개변수(Default Parameters)
- Async/Await
<br /><br />

## 알고리즘
- map
- queue
- stack
- 추가적으로 진행 중
<br /><br />
